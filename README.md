# Call Center Report Dashboard (Excel Project)
# Project Overview

This project presents an interactive Call Center Performance Dashboard built in Microsoft Excel to analyze call activity, performance, and customer satisfaction.
The dashboard allows managers to easily monitor KPIs such as total calls, revenue generated, call duration, ratings, and representative performance. It visually highlights trends and comparisons using charts, pivot tables, and slicers.

# Data Source

The dataset used for this project was taken from a YouTube tutorial on Excel Dashboard Design.
It contains information such as:

Call IDs

Representative IDs

City (Cincinnati, Cleveland, Columbus)

Call Amount

Call Duration

Customer Ratings

Gender

# Tools Used

Microsoft Excel

Power Pivot

Power Query

Pivot Tables & Pivot Charts

Data Model

DAX (Data Analysis Expressions)

# Data Cleaning

Performed in Power Query:

Removed duplicates and blanks

Standardized column names

Formatted numeric and date fields

Merged multiple datasets (Calls, Representatives, Ratings, and City Data)

Created relationships between tables in the Excel Data Model

Average Rating: 3.9 / 5
